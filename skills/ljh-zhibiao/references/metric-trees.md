# 内容电商指标树与公式

这份参考用于定位波动，不负责替平台定义字段。使用每条公式前，先核对分子、分母、时间范围、归因范围、含退或净成交口径。

## 1. 经营总树

### 成交与利润

```text
下单 GMV = 支付订单数 × 客单价
净 GMV = 下单 GMV × (1 - 退货率)
支付订单数 = 支付买家数 × 人均支付订单数
支付买家数 = 可成交流量 × 支付转化率
```

波动下钻顺序：

```text
净 GMV
  -> 下单 GMV / 退货率
  -> 支付订单数 / 客单价
  -> 可成交流量 / 支付转化率
  -> 对应平台的流量与转化子树
```

### 投放与盈亏

```text
ROI = 净 GMV ÷ 投放花费
费比 = 投放花费 ÷ 净 GMV
费比 = 1 ÷ ROI
毛利 = 净 GMV × 毛利率
渠道贡献 = 毛利 - 投放花费 - 佣金 - 已明确分摊的其他成本
盈亏平衡 ROI = 1 ÷ 毛利率
```

限定：费比与 ROI 的倒数关系只在两者使用同一 GMV、同一花费、同一归因窗口时成立。盈亏平衡 ROI 只覆盖毛利与投放花费的简化判断；有平台佣金、人力、履约等成本时，需要把这些成本带入总账。

## 2. 小红书图文树

### 主链路

```text
曝光
  -> 阅读：封面点击率
  -> 商品点击：商品点击率
  -> 支付买家：支付转化率
  -> GMV：客单价
  -> 净 GMV：退货率
```

在后台定义允许时，可写为：

```text
阅读数 = 曝光数 × 封面点击率
商品点击数 = 阅读数 × 商品点击率
支付买家数 = 商品点击数 × 支付转化率
下单 GMV = 支付买家数 × 客单价
净 GMV = 下单 GMV × (1 - 退货率)
```

在分母能够首尾衔接时：

```text
曝光到商品点击效率 = 封面点击率 × 商品点击率
曝光到支付效率 = 封面点击率 × 商品点击率 × 支付转化率
```

复合效率用于识别“封面点击上涨，但商品点击或支付链路整体变差”的情况。任一指标分母不同，就回到原始计数重算。

必须核对：商品点击率的分母可能是阅读、商品曝光或其他平台口径；支付转化率的分母可能是商品点击、访客或支付意向用户。以实际后台为准。

### 阅读价值与投放效率

```text
阅读价值 = 归因 GMV ÷ 阅读数
阅读成本 = 投放花费 ÷ 付费阅读数
ROI = 阅读价值 ÷ 阅读成本
```

最后一条只在阅读价值与阅读成本使用相同归因范围、且阅读数口径一致时成立。

### 诊断含义

| 第一个红灯 | 首先回答的问题 |
|---|---|
| 曝光下降 | 发布量、审核、投放量、账号状态或内容供给是否改变 |
| 封面点击下降 | 第一眼主视觉、标题承诺、人群和场景是否改变 |
| 商品点击下降 | 图 2-5、正文购买理由、商品链接承接是否改变；高点击入口是否吸引了错误人群或制造了错误期待 |
| 支付转化下降 | 价格、优惠、规格、信任、详情页和新客教育是否改变 |
| 客单下降 | SKU 结构、组合装、优惠门槛和新旧客结构是否改变 |
| 退货上升 | 人群预期、内容承诺、商品体验和履约是否错位 |

## 3. 抖音短视频树

先确认短视频的终点是商品成交，还是引导进入直播间。

### 短视频直接成交

```text
曝光
  -> 有效观看或内容点击
  -> 商品点击
  -> 加购或下单
  -> 支付
  -> 净 GMV
```

常用拆法：

```text
商品点击数 = 有效观看数 × 商品点击率
支付订单数 = 商品点击数 × 支付转化率
下单 GMV = 支付订单数 × 客单价
净 GMV = 下单 GMV × (1 - 退货率)
```

三秒完播、整体完播、互动和收藏用于判断内容拉人、留人和兴趣质量。它们是过程指标，不自动等于支付结果。

### 短视频引导直播

```text
短视频曝光
  -> 点击或进房
  -> 直播间停留
  -> 商品点击
  -> 加购
  -> 支付
  -> 净 GMV
```

素材 CTR 下跌优先查内容入口；CTR 正常但进房或停留下降，优先查素材承诺与直播间实况是否一致。

## 4. 抖音直播树

### 流量购买与进房

```text
曝光量 = 投放花费 ÷ CPM × 1000
进房人数 = 曝光量 × 进房率
商品点击人数 = 进房人数 × 商品点击率
加购人数 = 商品点击人数 × 加购率
支付买家数 = 加购人数 × 支付率
下单 GMV = 支付买家数 × 客单价
净 GMV = 下单 GMV × (1 - 退货率)
```

平台若使用不同分母，逐项替换为后台定义。平均停留时长、开播时长和场次是重要护栏：

- 开播时长、场次下降会直接压低可获得流量上限。
- 平均停留下降通常会削弱后续商品点击和支付样本，但它不是上述乘法链中的固定转化率。
- CPM 上涨会在花费不变时减少曝光量。

### 账户与素材前置层

```text
账户和计划状态
  -> 是否能花钱
素材 CTR 与首发周期
  -> 是否能拉人
直播间停留
  -> 是否能留人
点击 / 加购 / 支付
  -> 是否能转化
退货 / 秒退
  -> 净成交质量
```

## 5. 新客与获客树

```text
新客数 = 支付买家数 × 新客占比
CAC = 拉新花费 ÷ 新客数
首单毛利 = 新客首单净 GMV × 毛利率
单客首单毛利 = 首单客单价 × 毛利率
```

新客数下降时，同时拆支付买家数和新客占比。CAC 上升时，同时拆拉新花费和新客数。渠道归因不完整时，必须标注能识别与不能识别的流量。

## 6. 复购与客户价值树

复购必须按 cohort 观察：固定一批首购新客，给出明确观察窗口，再看其中有多少人回来购买。

```text
复购率 = 观察窗口内发生复购的新客数 ÷ 该 cohort 首购新客数
人均复购单数 = cohort 复购订单数 ÷ cohort 首购新客数
复购 GMV = cohort 首购新客数 × 人均复购单数 × 复购客单价
客户终身毛利 = 首单毛利 + 复购毛利
客户终身净贡献 = 客户终身毛利 - CAC
```

不要混用「当月买两次的人数占比」「当月发生复购的人数占比」和「某批新客未来一段时间复购率」。三者回答的问题不同。

复购下钻：

```text
复购率 / 人均复购单数
  -> 产品消耗周期与体验
  -> 复购 SKU 和组合装
  -> 回流通道
  -> 内容触达与新场景
  -> 价格、库存、履约
```

## 7. 内容供给与效率树

```text
内容测试量 = 进入真实发布或投放测试的素材数
素材达标率 = 达标素材数 ÷ 测试素材数
单条达标素材成本 = 本轮测试总成本 ÷ 达标素材数
内容贡献 GMV = 达标素材在统一归因口径下贡献的 GMV
```

达标线必须由团队按平台、阶段和经营目标定义。不要把播放、点赞或单一点击指标直接当作全链路达标。

因子与模板层：

```text
EV = 价值 × 胜率 ÷ 成本
```

- S 级因子：多次验证的顶级因子。
- A 级因子：已经验证，次数仍不足。
- 测试中因子：刚进入库，还没有稳定结论。
- 有效模板：用团队定义的复制次数、消耗、GMV、ROI 或其他结果门槛验收。
- 模板衰减：同一模板在可比投放条件下，达标率、消耗能力或结果贡献持续下降。

内容效率下降时，先拆测试量、达标率、单条达标成本、有效模板数和模板衰减，不要只看发布量。
